# PicoPlay

A beautifully designed collection of games. 

This platform focuses on minimal design, clean architecture, and a distraction-free user experience.

## Tech Stack

* **Framework:** Vue 3 (Composition API) + TypeScript
* **Build Tool:** Vite
* **Styling:** Tailwind CSS v4
* **State Management:** Pinia
* **Routing:** Vue Router 5

## Quick Start

```bash
# Install dependencies
npm install

# Start the development server
npm run dev

# Build for production
npm run build
